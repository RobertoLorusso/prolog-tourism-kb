
####### INFORMAZIONI SUI FILES ##########

All'interno del file rules.pl sono presenti le regole.
All'interno del file facts.pl sono presenti i fatti.
All'interno del file new_fact sono presenti le preferenze dell'utente, asserite durante l'esecuzione.

All'interno del file explanation.txt è presente la spiegazione dei risultati ottenuti dall'algoritmo. Alla fine del file è presente il path ottimale ottenuto. 


All'interno del file alternative-paths.txt è presente la spiegazione dei risultati ottenuti dall'algoritmo per la costruzione dei path alternativi. Per visualizzare tutti i path alternativi ottenuti è sufficiente cercare le parole chiave 'Alla fine' all'interno del file stesso.



######### ESECUZIONE ###########



Per eseguire i test è necessario consultare il file rules.pl e richiamare la regola main.


Per effettuare un test all'interno di bari si possono utilizzare le seguenti coordinate.

- Latitudine = 41.126427
- Longitudine = 16.872545



Per effettuare un test all'interno di firenze si possono utilizzare le seguenti coordinate.

- Latitudine = 43.7677883
- Longitudine = 11.2552763
